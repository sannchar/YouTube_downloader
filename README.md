# YouTube Downloader

A fast, modern desktop application for downloading videos or extracting audio (MP3) from YouTube. Built with Python, `customtkinter`, and `yt-dlp`.

## Features
- **Modern UI**: Dark/Light mode support with a clean, system-themed interface.
- **Fast Startup**: Instantly launches and is ready to use.
- **Video & Audio Support**: Download full MP4 videos or extract MP3 audio.
- **Auto Folder**: Automatically organizes downloaded files into a `Downloads` folder.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sannchar/YouTube_downloader.git
   cd YouTube_downloader
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. **(Required for MP3)**: Make sure you have [FFmpeg](https://ffmpeg.org/download.html) installed and added to your system PATH.

## Usage
Run the application using:
```bash
python main.py
```
Paste a YouTube link, select your desired format, and click "Download"!
